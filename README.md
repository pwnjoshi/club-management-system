# Unified College Club Management System

## Overview

A full-stack web application to manage college clubs, events, attendance, and certificate verification.

## Tech Stack

- Frontend: React
- Backend: Spring Boot (Java)
- Database: MySQL
- Authentication: JWT

- Event Management
- Event Registration
- Attendance Tracking
- Certificate Generation
- Certificate Verification

- /frontend -> React UI
- /docs -> Documentation
